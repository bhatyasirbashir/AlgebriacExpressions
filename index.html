Basic or Main algebraic Expressions

  1. (a+b) 2 = a 2 +b 2 + 2ab

    Now,let is derive the above identity or where does this identity come from that is (a+b) 2 = a 2 +b 2 + 2ab
    We are given (a+b) 2 = a 2 +b 2 + 2ab
    (a+b) 2 = (a+b) (a+b)
    (a+b) 2 = a(a+b) + b(a+b)
    (a+b) 2 = a 2 + ab + ba + b 2
    (a+b) 2 = a 2 + 2ab + b 2
    (a+b) 2 = a 2 + b 2 + 2ab
    Hence, (a+b) 2 = a 2 +b 2 + 2ab

  2. (a+b) 2 = a 2 +b 2 - 2ab

    Now,let is derive the above identity or where does this identity come from that is (a+b) 2 = a 2 +b 2 - 2ab
    We are given (a-b) 2 = a 2 +b 2 - 2ab
    (a-b) 2 = (a-b) (a-b)
    (a-b) 2 = a(a-b) - b(a-b)
    (a-b) 2 = a 2 - ab - ba + b 2
    (a-b) 2 = a 2 - 2ab + b 2
    (a-b) 2 = a 2 + b 2 - 2ab
    Hence, (a-b) 2 = a 2 +b 2 - 2ab

  3. (a+b) (a-b) = a 2 -b 2

    Now,let is derive the above identity or where does this identity come from that is (a+b) (a-b) = a 2 -b 2
    We are given (a+b) (a-b) = a 2 -b 2
    (a+b) (a-b) = a(a-b) +b(a-b)
    (a+b) (a-b) = a 2 -ab + ba - b 2
    (a-b) (a-b) = a 2 - ab + ab - b 2 [because ab=ba]
    (a-b) (a-b) = a 2 - b 2 [because -ab+ba=0]
    Hence, (a+b) (a-b) = a 2 -b 2

  4. (x+a) (x+b) = x 2 + (a+b)x + ab

    Now,let is derive the above identity or where does this identity come from that is (x+a) (x+b) = x 2 + (a+b)x + ab
    We are given (x+a) (x+b) = x 2 + (a+b)x + ab
    (x+a) (x+b) = x(x+b) + a(x+b)
    (x+a) (x+b) = x 2 + xb + ax + ab
    (x+a) (x+b) = x 2 + (a+b)x + ab
    Hence, (x+a) (x+b) = x 2 + (a+b)x + ab
